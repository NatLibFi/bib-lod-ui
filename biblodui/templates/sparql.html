SPARQL access

This is the SPARQL endpoint of the National Library of Finland Linked Data Service. You can query our linked data sets either using the YASGUI-based form below, or by using this endpoint directly in any program that can perform SPARQL 1.1 queries.

The URL for this endpoint is https://data.nationallibrary.fi/bib/sparql.

See the data model documentation for more information about what kind of data is available in this endpoint.

Note: The endpoint has a timeout of 30 seconds to protect the server from overloading. If you need to perform heavy queries, consider using the Linked Data Fragments endpoint or downloading data dumps for local use.
